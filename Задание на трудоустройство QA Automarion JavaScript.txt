Задание на трудоустройство QA Automarion JavaScript


1. Послать GET запрос на https://api.rabota.ua/dictionary/city, найти в респонсе нужный объект по ключу "en" со значением  "Romny".
2. В проекте должен быть заготовленный JSON-файл с шаблоном ожидаемого объекта с пустыми значениями ключей centerId и id, формата:
{centerId: "", 
centerName: null, 
id:"" , 
ru: "Ромны", 
ua: "Ромни", 
en: "Romny"}
3. Из полученного и отфильтрованного из целого респонса объекта нужно заполнить пустые значения с соответствующими ключами в шаблоне, чтобы в итоге получилось 2 идентичных объекта.
4. Проверить идентичность с помощью сравнения объектов.


Задание предпочтительно выполнить на ES5/6 с использованием его свойств для синхронности выполнения кода.
